# Nick Name Web Component

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://opensource.org/licenses/MIT)

## Description

The nick-name web component replaces a name with a random nickname from a list of predefined nicknames.

## Usage

```html
A nickname for John is <nick-name>Jack</nick-name>
<script src="nick-name.js"></script>
```
